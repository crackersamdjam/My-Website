Programming Information

My main language is C++, which I use for competitive programming and robotics. I also have experience with Java and Python programming. I use Bash, C, and Python for interfacing and scripting. Finally, I have experience with HTML, CSS, and JavaScript, all of which I used to create this website.

I write online contests regularly on Atcoder, Codeforces (thanks Mike MikeMirzayanov Mirzayanov for the great Polygon and Codeforces platforms), and DM::OJ. I also compete in the Facebook Hacker Cup, Google's Coding Competitions, and many more.
